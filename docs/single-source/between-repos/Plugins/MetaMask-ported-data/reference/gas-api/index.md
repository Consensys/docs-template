# Gas API

The Gas API is a tool that delivers real-time gas prices for supported networks, enabling users to identify the best times to execute transactions based on current rates and intricacies introduced by [EIP-1559](https://eips.ethereum.org/EIPS/eip-1559).

:::info
See the list of supported Gas API networks.
:::
